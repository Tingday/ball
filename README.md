<h3>2019-02-23 玩法说明：</h3>
<pre>
1.通过A/D来控制挡板的移动，保证弹珠球不掉落；
2.当挡板移动时，弹珠球的速度会改变，甚至有时候会突然间加快；
3.尽量通过消除砖块获得更高的分数。
</pre>
